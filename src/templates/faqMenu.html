FAQ

Here are some frequently asked questions about GDSearch.

What is GDSearch?

GDSearch is a web application that allows you to explore user data from the game Geometry Dash. You can search for a user, view their information, and see the leaderboard of the top players in the game.

How do I search for a user?

To search for a user, go to the Search menu and enter the username of the player you want to find. Click the Search button, and you will be directed to the user's information page.

How do I view the leaderboard?

To view the leaderboard, click on the Leaderboard tab in the navigation bar. You can sort the leaderboard by stars, diamonds, or user coins by selecting the desired filter from the dropdown menu.

How do I pin a user?

To pin a user, search for the user you want to pin and click the Pin button on their information page. The user will be added to your pinned users list on the Search menu for easy access.
