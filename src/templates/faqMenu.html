FAQ

Here are some frequently asked questions about GDSearch.

What API does GDSearch use?

The "GDBrowserAPI" by GDColon

What endpoints are used?

https://gdbrowser.com/api/leaderboard : Leaderboard

https://gdbrowser.com/api/profile/ + Username : User Information

https://gdbrowser.com/api/level/ + Level ID : Level Information

What are the roots in this website?

http://localhost:8080/mainMenu : Main menu.

http://localhost:8080/searchMenu : Search menu.

http://localhost:8080/findUser : User stat menu.

http://localhost:8080/findLevel : Level stat menu.

http://localhost:8080/faqMenu : FAQ menu (you are here).

http://localhost:8080/leaderboard : Leaderboard menu.

http://localhost:8080/leaderboard/subtractPage : Previous Page.

http://localhost:8080/leaderboard/addPage : Next Page.

http://localhost:8080/pinUser : Pin selected user

http://localhost:8080/unPinUser : Unpin selected user
